# Book of Set Health

Infinite Inner World shared metric reference

Folder: the book of complexity

Scope:

```text
Compressed full-set health identity
Natural draw pressure tone + key sorted body parts
```

## Purpose

Set Health is the readable full-set health phrase.

It answers:

```text
What is the set's natural pressure tone?
Where does the set begin?
How does the middle of the sorted body behave?
Where does the set end?
```

Set Health is intentionally compressed. It gives a fast full-set read without forcing every script to print every classification label.

## Formula

Set Health uses this order:

```text
Natural Draw Pressure Tone
+ S1 Core Band
+ S2->S3 Relation
+ S3->S4 Relation
+ S5 Endpoint Band
```

In field form:

```text
set_health =
pressure_tone
+ core_band
+ middle_entry_bridge
+ middle_bridge_exit
+ endpoint_band
```

## Why Draw Pressure Leads

The first word comes from Draw Pressure, not Sorted Pressure.

Reason:

```text
Draw Pressure is the set in its natural state.
```

So the health phrase begins with the natural pressure tone.

Sorted form supplies the stable body landmarks:

```text
S1 Core
S2->S3 middle relation
S3->S4 middle relation
S5 Endpoint
```

## What Set Health Gives You

Set Health gives a compressed read of the whole body:

```text
natural draw pressure tone
starter/core identity
middle relation identity
endpoint identity
```

For most surface reporting, Set Health is the cleanest full-set label.

## What Set Health Does Not Give You

Set Health does not show:

```text
exact numbers
exact gaps
draw route
motion energy
pressure fusion
technical anatomy
set style signature
```

Those live in Apex Point, Draw Style, Draw Motion, Pressure Fusion, and Set Anatomy.
