# Book of Apex Point

Infinite Inner World shared metric reference

Folder: the book of complexity

Scope:

```text
Full five-number set
Draw form + Sorted form neutral
```

## Purpose

Apex Point is the shared numeric weight of the full set before the set is read as Sorted form or Draw form.

It answers:

```text
How heavy is the whole set?
How wide is the set from smallest to largest?
What is the shared numeric ceiling/floor pressure?
```

Apex Point is not a lane metric. It belongs to the whole five-number body.

## Total

Total is the complete set sum.

Formula:

```text
total = D1 + D2 + D3 + D4 + D5
```

Because Draw form and Sorted form contain the same five numbers, the total is shared.

Example:

```text
draw form   49-53-3-26-61
sorted form 3-26-49-53-61

total = 49 + 53 + 3 + 26 + 61
total = 192
```

## Finality Point

FP means Finality Point.

Formula:

```text
fp = largest_number - smallest_number
```

Example:

```text
smallest_number = 3
largest_number  = 61

fp = 61 - 3
fp = 58
```

FP is the full-set spread. It tells how far the body stretches from bottom to top.

## Olympus Gauge

Apex gauges use Olympus labels. They do not borrow motion labels.

The Olympus cycle:

```text
1-2   Zeus
3-4   Hera
5-6   Poseidon
7-8   Demeter
9-10  Athena
11-12 Apollo
13-14 Artemis
15-16 Ares
17-18 Aphrodite
19-20 Hephaestus
21-22 Hermes

```

The cycle repeats every 24 points.

That means totals above 24 continue forward through the same two-number rhythm.

Example:

```text
191-192 Dionysus
```
