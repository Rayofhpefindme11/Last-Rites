# Book of Basics

This book is the baseline numeric-range reference for the Sorted form side of Infinite Inner World.

It covers the fixed sorted lanes, the 1-2 number-band identities, and the 35-class range systems used by gaps, Set Arc, and Set Relations. These ranges are the source of truth for rebuilding foundation scripts, predictive scripts, and RD findings without rediscovering the same classifications.

## Sorted Form

Sorted form always reads the five white balls in ascending order.

```text
S1 Core
S2 Entry
S3 Bridge
S4 Exit
S5 Endpoint
```

Sorted lane range rules:

```text
S1 Core     1-65
S2 Entry    2-66
S3 Bridge   3-67
S4 Exit     4-68
S5 Endpoint 5-69
```

These ranges are lane-validity boundaries. They do not replace the number-band identity of each number.

## Zero Handling

Sorted form cannot produce a zero gap between two valid white balls because duplicate white balls are invalid.

Zero labels exist for some wider metric systems, but Sorted basics are built around adjacent distinct numbers and therefore use the 1-2 through 69-70 range classes.
