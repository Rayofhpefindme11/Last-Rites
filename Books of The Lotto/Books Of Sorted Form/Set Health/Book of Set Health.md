# Book of Set Health

This book documents Set Health for Infinite Inner World.

Set Health is the compact readable identity of the full set. It combines the natural draw-order pressure tone with the sorted body identity.

## Purpose

Set Health answers this question:

```text
What is the clean readable health identity of this set?
```

It is intentionally short. It should be readable enough to scan without needing every lower-level field.

The deeper technical body belongs to Set Anatomy.

## Formula

Set Health formula:

```text
Set Health = Natural Draw Pressure Tone + S1 Core Band + S2->S3 Relation + S3->S4 Relation + S5 Endpoint Band
```

The first part comes from Draw Pressure because draw order is the set in its natural state.

The remaining parts come from Sorted form because Sorted form gives the clean body:

```text
S1 Core
S2->S3 middle relation
S3->S4 middle relation
S5 Endpoint
```

## Set Health Fields

Set Health contains:

```text
set_health
pressure_tone
core_band
middle_entry_bridge
middle_bridge_exit
endpoint_band
rule
```

Field meanings:

```text
set_health            full readable title
pressure_tone         natural draw pressure type converted into readable form
core_band             S1 number band
middle_entry_bridge   S2->S3 relation class in readable form
middle_bridge_exit    S3->S4 relation class in readable form
endpoint_band         S5 number band
rule                  formula statement
```

## Pressure Tone

Pressure tone comes from Draw Pressure, not Sorted Pressure.

Reason:

```text
Sorted Pressure always reads the organized set.
Draw Pressure reads the natural draw-order face.
```

Sorted form normally rises:

```text
S1 < S2 < S3 < S4 < S5
```

So Sorted Pressure normally becomes:

```text
+++ UPLIFT
```

That would make Set Health start with `Uplifting` too often. Draw Pressure gives the real natural formation tone.

## Sorted Body Parts

Set Health uses these sorted body parts:

```text
S1 Core Band
S2->S3 Relation
S3->S4 Relation
S5 Endpoint Band
```

It does not include:

```text
S1->S2 relation
S4->S5 relation
middle span
middle zone
edge pressure
full relation
```

Those are part of Set Anatomy.

## What Set Health Tells You

Set Health gives a compact identity:

```text
natural tone
starting sorted identity
middle relationship identity
ending sorted identity
```

Example:

```text
Tensioned Sky Fighter High Kid High King
```

This tells us:

```text
The natural draw face is Tensioned.
The sorted core starts in Sky.
The first middle relation is Fighter High.
The second middle relation is Kid.
The endpoint lands in High King.
```

## What Set Health Does Not Tell You

Set Health does not fully describe:

```text
Set Style grouping
entry edge
exit edge
middle span
middle zone
edge balance
edge pressure
full set relation
technical signature
```

For a full set read, use:

```text
Set Health
Set Style
Set Anatomy
```

## Relationship To Set Anatomy

Set Health is the readable title.

Set Anatomy is the technical body.

Set Health should stay compact. Do not overload it with every field.

Set Anatomy should carry the deeper details that scripts need for foundation and predictive work.
