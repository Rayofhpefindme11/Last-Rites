## Shared Metric Order

The shared metrics should be read in this order:

```text
1. Apex Point
2. Set Pressure Bridge
3. Pressure Fusion
4. Set Health
5. Set Anatomy
6. Full Set Packet
```

Reason:

```text
Apex Point gives the set's numeric weight.
Set Pressure Bridge joins sorted pressure and draw pressure.
Pressure Fusion names the pressure redistribution.
Set Health gives the compact full-set identity.
Set Anatomy expands that identity into technical structure.
Full Set Packet shows the complete IIW body.
```
